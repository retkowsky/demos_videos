# Azure AI containers - demo videos

This repository contains short **demo videos** showcasing **Azure AI** scenarios (including Azure AI Document Intelligence and translation demos), intended for learning, presentations, and quick walkthroughs.

## Contents

Videos currently included at the repository root:

- `Démo 1 Azure AI Document Intelligence + slm.mp4`
- `Démo 2 Traduction de texte.mp4`
- `Démo 3 Traduction de voix.mp4`
- `Démo 4 Traduction vocale de fichiers.mp4`
- `Démo 5 Recherche visuelle.mp4`
- `Demo-Traducteur.mp4`

## How to view

### Option A — View on GitHub (recommended)
1. Open the repository: `retkowsky/Azure_AI_containers_demos_videos`
2. Click a `.mp4` file in the file list
3. Use the embedded player to stream the video (or download it)

### Option B — Clone and watch locally
```bash
git clone https://github.com/retkowsky/Azure_AI_containers_demos_videos.git
cd Azure_AI_containers_demos_videos
```

Then open any `.mp4` file with your preferred media player.
